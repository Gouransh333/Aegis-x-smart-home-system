# Aegis-x-smart-home-system
**AEGIS-X: Smart Habitat Monitoring and Alert System**!

Over the past few weeks, I worked on designing and developing an IoT-based monitoring system capable of detecting and responding to real-time environmental and security-related events. The system continuously monitors fire, rain, motion, and emergency situations and provides instant alerts through LEDs, a buzzer, and a live monitoring dashboard.

🔧 **What I Built**
• Real-time Fire Detection System
• Rain Monitoring and Alert System
• Motion Detection using IR Sensor
• Emergency Alert System using Touch Sensor
• Smart LED and Buzzer-based Alert Mechanism
• Interactive Web Dashboard for Live Monitoring and Visualization

💻 **Technologies & Tools Used**
• NodeMCU ESP8266
• Flame Sensor, Rain Sensor, IR Sensor, Touch Sensor
• Arduino IDE & Embedded C/C++
• HTML, CSS, JavaScript
• Chart.js for Data Visualization
• ESP8266 Web Server
• Wi-Fi Communication & JSON Data Exchange
• Python HTTP Server for Dashboard Hosting

🎯 **Key Outcomes Achieved**
✅ Real-time sensor monitoring and alert generation
✅ Successful integration of hardware and software components
✅ Live dashboard displaying sensor status, threat levels, and alert history
✅ Practical implementation of IoT concepts and wireless communication
✅ Development of a functional prototype capable of environmental monitoring and threat detection
